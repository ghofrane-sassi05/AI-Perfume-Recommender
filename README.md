# Parfum Expert AI

An intelligent perfume recommendation system designed to help users discover fragrances that match their personality, preferences, and olfactory tastes.

## Features

* AI-powered perfume recommendations
* Interactive and modern user interface
* Fragrance profile analysis
* Personalized suggestions
* Responsive design
* Fast performance with React and Vite

## Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* Framer Motion

### Backend

* Express.js
* Node.js

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/parfum-expert-ai.git
```

Navigate to the project folder:

```bash
cd parfum-expert-ai
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```
